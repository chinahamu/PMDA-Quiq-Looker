# Icon Assets / アイコン素材

**English:** This directory contains the source icon draft and notes used for the released extension and future store updates.

**日本語:** このディレクトリには、リリース済み拡張機能と今後のストア更新で使用するアイコン原稿とメモを格納しています。

## Files / ファイル

- `pmda-quick-looker-icon.svg`: editable source icon draft. / 編集可能なアイコン原稿。

## Chrome Web Store assets / Chrome ウェブストア素材

**English:** For store updates, export the SVG to PNG files and confirm they render sharply.

**日本語:** ストア更新時は SVG を PNG に書き出し、鮮明に表示されることを確認してください。

Recommended outputs / 推奨出力:

- `icon-128.png` for Chrome Web Store submission. / Chrome ウェブストア提出用。
- `icon-48.png`, `icon-32.png`, and `icon-16.png` for the extension package. / 拡張機能パッケージ用。

## Visual concept / ビジュアルコンセプト

- Magnifying glass: quick lookup. / 虫眼鏡: 素早い検索。
- Document sheet: package insert reference. / 書類: 添付文書参照。
- Blue productivity palette. / 青系の生産性パレット。
