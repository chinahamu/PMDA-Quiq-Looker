# Support PMDA Quick Looker / PMDA Quick Looker を支援する

## English

PMDA Quick Looker is maintained as an open-source Chrome extension. If the project is useful to you or your organization, you can support ongoing maintenance through the GitHub Sponsors button configured in `.github/FUNDING.yml`.

Support is voluntary. It does not change the Apache License 2.0 terms, does not create a paid support contract, and does not affect the handling of security issues or bug reports.

### What support helps with

- Maintenance of Chrome extension compatibility.
- Documentation updates for Chrome Web Store publication.
- Test coverage, accessibility improvements, and release checks.
- Continued privacy-first development that avoids patient data collection.

### How to enable or confirm the button

1. Keep `.github/FUNDING.yml` in the repository.
2. Confirm that it contains the maintainer's GitHub Sponsors account.
3. After the repository is public, GitHub shows the Sponsor button when the account and repository are eligible.
4. Keep this document linked from `README.md` and `docs/index.md`.

## 日本語

PMDA Quick Looker は OSS の Chrome 拡張機能としてメンテナンスされています。このプロジェクトが個人または組織の役に立つ場合、`.github/FUNDING.yml` で設定された GitHub Sponsors ボタンから継続的なメンテナンスを支援できます。

支援は任意です。Apache License 2.0 の条件は変わらず、有償サポート契約を意味するものでもありません。また、セキュリティ課題や不具合報告の扱いに影響しません。

### 支援が役立つこと

- Chrome 拡張機能としての互換性維持。
- Chrome ウェブストア公開に向けたドキュメント更新。
- テスト整備、アクセシビリティ改善、リリース確認。
- 患者情報を収集しないプライバシー重視の開発継続。

### ボタンを有効化・確認する手順

1. `.github/FUNDING.yml` をリポジトリに残します。
2. メンテナの GitHub Sponsors アカウントが記載されていることを確認します。
3. リポジトリを public にした後、アカウントとリポジトリが条件を満たすと GitHub 上で Sponsor ボタンが表示されます。
4. この文書を `README.md` と `docs/index.md` からリンクし続けます。
