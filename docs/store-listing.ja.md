# Chrome ウェブストア掲載文 / Chrome Web Store Listing

## 日本語

### 拡張機能名

PMDA Quick Looker

### 1行説明

選択した薬剤名から添付文書PDFリンクと関連する薬剤情報を素早く検索・参照できるChrome拡張機能です。

### 詳細説明

PMDA Quick Looker は、Web画面で表示されている薬剤名から、添付文書PDFリンクや関連する薬剤情報を素早く検索・確認するためのChrome拡張機能です。

主な機能:

- Webページ上で薬剤名を選択し、右クリックメニューから検索
- ポップアップから薬剤名を直接検索
- Chrome Side Panelで検索結果を確認
- 候補一覧、添付文書PDFリンク、取得できた薬剤情報を表示
- Side Panelでブックマーク、表示モード、文字サイズを設定
- 検索結果キャッシュをユーザーのブラウザ内に保存

### 言語

- 日本語
- 英語

### キーワード候補

PMDA, 添付文書, 医薬品, 薬剤検索, 電子カルテ, Chrome拡張, package insert, drug information, Japanese drug label

### スクリーンショット構成

1. コンテキストメニューから薬剤名を検索する画面
2. ポップアップで候補一覧を表示する画面
3. Side PanelでPDFリンクと薬剤情報を確認する画面
4. プライバシー/権限説明を補足する画面

### データ利用に関する説明

本拡張機能は、ユーザーが明示的に検索操作を行った薬剤名を、検索結果取得のために `drugwiki.meta-alchemist.co.jp` および `www.pmda.go.jp` へ送信することがあります。検索結果キャッシュ、ブックマーク、表示設定はユーザーのブラウザ内に保存されます。`drugwiki.meta-alchemist.co.jp` の標準的なAPIサーバーログには、セキュリティ監視、不正利用防止、不具合調査のため、IPアドレス、User-Agent、アクセス時刻、リクエストURL、HTTPステータス、検索キーワードが含まれる場合があります。患者氏名、カルテ番号、画面全文、閲覧履歴、広告ID、解析IDは収集しません。

## English

### Extension name

PMDA Quick Looker

### Short description

Quickly search package insert PDF links and related drug information from selected drug names.

### Detailed description

PMDA Quick Looker helps users search package insert PDF links and related drug information from drug names displayed in web pages.

Key features:

- Search selected drug-name text from the right-click context menu.
- Search manually from the extension popup.
- Review search results in the Chrome Side Panel.
- Display candidate records, package insert PDF links, and available drug information.
- Configure Side Panel bookmarks, display mode, and font-size preferences.
- Store search-result cache data in the user's browser.

### Data usage explanation

When the user explicitly starts a lookup, the extension may send the selected or entered drug name to `drugwiki.meta-alchemist.co.jp` and `www.pmda.go.jp` to retrieve search results and package insert PDF links. Search-result cache data, bookmarks, and display settings are stored locally in the user's browser. Standard API server logs on `drugwiki.meta-alchemist.co.jp` may include IP address, user agent, access time, request URL, HTTP status, and search keyword for security monitoring, abuse prevention, and troubleshooting. The extension does not collect patient names, chart IDs, full page contents, browsing history, advertising IDs, or analytics IDs.
