PMDA Quick Looker Privacy Policy / プライバシーポリシー

Last updated / 最終更新日: 2026-06-26

English

PMDA Quick Looker is a Chrome extension that helps users search package insert PDF links and related drug information.

Information used

The extension may use selected or entered drug-name text, search keywords, search results, package insert PDF URLs, drug information, bookmarks, display settings, and local error information.

Users should not select or enter patient names, chart IDs, or other patient-identifying information as search keywords.

Storage

Search-result cache data, bookmarks, and display settings are stored in the user's browser.

External requests

The extension may send lookup requests to drugwiki.meta-alchemist.co.jp and www.pmda.go.jp. The selected or entered drug-name text is sent as a search keyword to retrieve package insert candidates and related drug information.

API server logs

The drug search API may keep standard server logs for security monitoring, abuse prevention, and troubleshooting. These logs may include IP address, user agent, access time, request URL, HTTP status, and the search keyword included in the request. Logs are not sold to advertisers or data brokers.

Data not collected

The extension does not collect patient names, chart IDs, full page contents, browsing history, advertising identifiers, analytics identifiers, or Google Analytics data.

Contact

Please use GitHub Issues for bug reports and privacy questions when the report does not contain sensitive information. Use the maintainer's public contact channel when private details are required.

日本語

PMDA Quick Looker は、添付文書PDFリンクと関連する薬剤情報の検索を補助するChrome拡張機能です。

利用する情報

選択または入力された薬剤名、検索キーワード、検索結果、添付文書PDF URL、薬剤情報、ブックマーク、表示設定、ローカルエラー情報を利用することがあります。

検索キーワードとして、患者氏名、カルテ番号、その他の患者識別情報を選択・入力しないでください。

保存

検索結果キャッシュ、ブックマーク、表示設定はユーザーのブラウザ内に保存されます。

外部リクエスト

検索のために drugwiki.meta-alchemist.co.jp と www.pmda.go.jp へリクエストを送信することがあります。選択または入力された薬剤名は、候補一覧と関連する薬剤情報を取得するための検索キーワードとして送信されます。

APIサーバーログ

薬剤検索APIでは、セキュリティ監視、不正利用防止、不具合調査のために標準的なサーバーログを保存することがあります。ログには、IPアドレス、User-Agent、アクセス時刻、リクエストURL、HTTPステータス、検索キーワードが含まれる場合があります。ログは広告会社やデータブローカーへ販売しません。

収集しない情報

患者氏名、カルテ番号、ページ全文、閲覧履歴、広告識別子、解析識別子、Google Analyticsデータは収集しません。

お問い合わせ

不具合報告やプライバシーに関する質問は、機微情報を含まない場合は GitHub Issues、非公開の詳細が必要な場合はメンテナの公開連絡先を利用してください。
